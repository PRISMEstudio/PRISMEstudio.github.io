Téléchargement






Windows


Après avoir télécharger le fichier "PRISMELauncher_In Depth_Setup_Win.exe" vous devrez:

  • Executer le programme en tant que Administrateur.
  • Passer le SmartScreen ou le pare-feu pour pouvoir utiliser le launcher.
  • Installer le launcher.
  • Télécharger le jeu et les mises à jours.
  • Cliquer sur "Play".
  • Installer les composant de base du moteur (Unreal Engine) nécessaire pour faire fonctionner le jeu.
  • Puis amusez-vous !
