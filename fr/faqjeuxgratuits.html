Conditions d'utilisations

#Jeux Gratuits est un service gratuit administré par PRISME Studio.

Nous ne sommes en aucun cas responsable des offres de jeux gratuits, celles-ci sont proposé sur différentes plateformes applicants chacunes leurs propre politique.

Il est interdit de recopier et/ou de modifier le contenu ainsi que les images des offres. (voir cette section)

Les images utilisées afin d'illustrer les offres appartiennent a leurs développeur(s)/éditeur(s) respectif.

PRISME Studio,
Tout droit réservé.

Conditions de participation aux giveaway

Vous devez simplement répondre aux critères suivant:

  • - Avoir un compte discord avec le paramètre "Autoriser les messages privés venant des membres du serveur" d'activé.
    Clic droit sur le serveur > Paramètre de confidentialité
  • - Un compte sur la plateforme correspondante au giveaway, afin de réclamer votre récompense.
  • - Lire les règles ci-dessous.

Compétition / Giveaway Règlement

Merci de lire attentivement le document:
Compétition / Giveaway Règlement 13/11/2021(pdf)

Comment accéder aux annonces de jeux gratuits ?

Rendez-vous sur le site https://prismestudio.github.io/jeuxgratuits et cliquez sur le bouton "Connect" sur le widget Discord.
Quand vous êtes sur le serveur Discord, sélectionnez votre langue dans le salon #welcome.

Comment mettre les annonces sur mon serveur ?

Pour mettre les annonces dans votre serveur, veuillez suivre les instruction ci-dessous.

1- Aller dans le salon #jeux-gratuits.

2- En haut de la fenêtre, à-côté du titre du salon, cliquez sur le bouton "Suivre".


3- Sélectionnez le serveur ainsi que le salon où vous souhaitez ajouter les annonces.

Si vous avez un problème n'hésitez pas à nous contacter via la rubrique "Nous contacter" en mettant dans le message votre compte discord (pseudo#0000), toutes ces informations seront bien sûr gardées confidentielles.
