v.0.0.1.025

News:
- Added an end-of-level screen, containing information on the previously played map, statistics, as well as a summary of the rewards you won.
- Added a simple enemy that comes towards the player, in the final room of the demo map.
- Addition of a button to access the news panel in the main menu.
- Improvement of the news panel, allowing now to have several slides and navigate between them using the arrows.
- Added animation for the credits screen as well as for the patch note, fixed bug, and level selection screens.
- Changed the application launch cinematics.
- The usage warning when launching the application is now translated into the different languages of the game.
---
Bug fixed:
- Fixed a bug that didn't keep the gold added to the character in memory.
---
Other:
- Stop support for the following platform: Windows 32bits. (Additional Info: Windows 64bits version is still supported by In Depth).
- Upgrade to Unreal Engine version 4.27.
