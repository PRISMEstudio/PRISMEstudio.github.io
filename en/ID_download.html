Download






Windows


After you have download the file "PRISMELauncher_In Depth_Setup_Win.exe" you should:

  • Run the program as Administrator.
  • Pass the Smarscreen or the firewall to be able to use the launcher.
  • Install the launcher.
  • Download the game and all updates.
  • Click on "Play".
  • Install the basic engine components (Unreal Engine) required to run the game.
  • Then have fun!
