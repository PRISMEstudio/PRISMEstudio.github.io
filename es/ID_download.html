Descargar






Windows


Después de descargar el archivo "PRISMELauncher_In Depth_Setup_Win.exe" debería:

  • Ejecute el programa como administrador.
  • Pase SmartScreen o el cortafuegos para poder usar el lanzador.
  • Instale el lanzador.
  • Descarga el juego y las actualizaciones.
  • Haga clic en "Play".
  • Instale los componentes básicos del motor (Unreal Engine) necesarios para ejecutar el juego.
  • ¡Entonces diviértete!
